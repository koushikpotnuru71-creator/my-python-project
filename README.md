# GitHub Fork Workflow Practice

This repository demonstrates a basic GitHub fork workflow using Git and GitHub.

The goal of this project is to practice common Git operations that are used in real development and DevOps workflows.

---

## Project Purpose

This project was created to practice contributing to repositories using the GitHub fork workflow.

The workflow includes:

- Forking a repository
- Cloning a fork locally
- Creating a branch
- Making changes
- Committing changes
- Pushing a branch
- Creating a Pull Request
